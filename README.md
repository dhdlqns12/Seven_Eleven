## 유니티 입문 팀 과제 프로젝트 : 레퍼런스 게임을 바탕으로 우리만의 게임 제작하기!

## 1. 프로젝트 소개
SevenEleven은 유니티 2D플랫포머 퍼즐 게임으로 플레이어가 두 캐릭터를 조종해 맵에 배치된 장애물을 피하고 사물을 이용해 도착지에 도달하는 게임입니다.
두 캐릭터를 적절히 이용해 퍼즐을 풀어 도착지에 도달해 보세요!

### **개발 환경**
- **Engine**: Unity 2022.3.62f2
- **Language**: C# 
- **IDE**: Visual Studio, Rider
